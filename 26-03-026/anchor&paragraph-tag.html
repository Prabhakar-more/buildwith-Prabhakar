Attributes of target

_self (default) : Opens the link in the same tab/window.

_blank : Opens the link in a new tab or window.

_parent : Opens the link in the parent frame (used with frames/iframes)

_top : Opens the link in the full body of the window, breaking out of all frames

Custom Frame Name : You can target a specific iframe or window by name.

Anchor tag is used to link one page to another page/external resources we use 'a' tag for anchor. it has an attribute href (hyper reference) contains value 'path to redirect'
